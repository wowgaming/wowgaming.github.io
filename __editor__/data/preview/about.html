About WowGaming

WoWGaming  is a project that aims to provide a free and no-profit wiki, database and community spaces subdivided by expansion. Please read this page to understand why and how we can do it.
This project can be helpful for:

  • Create an historical archive of World of Warcraft legacy data
  • Discussing and Blogging about newest and oldest WoW Facts 



If you're interested in this project and would contribute
please, contact us on Discord
